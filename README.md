# Think Media Website

**Think Media** is a professional media and creative agency based in Uganda, offering:

- Photography (events, portraits, branding)
- Videography (ads, music videos, reels)
- PR & Content Creation (social media campaigns, brand storytelling)
- Video Editing and Post-Production

This repository contains the complete source code for the official **Think Media website**, including HTML, CSS, and assets.
